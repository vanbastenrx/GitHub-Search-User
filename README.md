# GitHub-Search-User
Uma tela simples com HTML CSS e JS puro usando a API do github para buscar usuários - <link>https://api.github.com/users/</link>

<br>

<strong>É só digitar o @usuário no campo de pesquisa e pronto, bem simples.</strong>
